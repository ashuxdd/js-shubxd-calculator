# Basic Calculator

A simple web-based calculator built using HTML, CSS, and JavaScript.  
It supports basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

- Clickable number and operator buttons
- Supports +, -, *, / operations
- Clear button to reset the input
- Backspace button to delete the last character
- Basic error handling for invalid expressions and division by zero

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6)

## How to Run

1. Clone the repository or download the ZIP.
2. Open `index.html` in any modern web browser.
3. Use the on-screen buttons to perform calculations.

## Project Structure

- `index.html` – main page containing the calculator layout
- `style.css` – styles for the calculator UI
- `script.js` – JavaScript logic for handling button clicks and calculations
